# 📋 TaskFlow - React Task Management App

TaskFlow is a feature-rich task management application built with React, designed to help users stay organized and track their tasks efficiently. It includes advanced features like filtering, categories, priority levels, due dates, animations, and dark mode.

---

## 🚀 Features

✅ User Login (Name stored with localStorage)  
✅ Add, Edit, Delete Tasks  
✅ Mark Tasks as Completed or Pending  
✅ Search Tasks by Title or Description  
✅ Filter Tasks by Status (All, Completed, Pending)  
✅ Filter Tasks by Category (Work, Personal, Shopping, Others, Uncategorized)  
✅ Priority Levels for Tasks (Low, Medium, High)  
✅ Due Dates for Tasks with Overdue Indication  
✅ Smooth Animations for Enhanced UI Experience  
✅ Dark Mode Toggle  
✅ Color-Coded Category Badges  
✅ Logout Button  
✅ Data Persistence via `localStorage`

## 🖥️ How to Run Locally

1. Clone the repository  
2. Navigate to project folder  
3. Run `npm install`  
4. Run `npm start`  
5. Open `http://localhost:3000` in your browser
